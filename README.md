# Machine Learning Portfolio

This repository is a comprehensive collection of my work in **Artificial Intelligence**, focusing on **Natural Language Processing (NLP)** and **Computer Vision (CV)**. It demonstrates a progression from foundational machine learning to state-of-the-art Deep Learning architectures.

---

## Featured Projects

### 1. [Smart Resume Parser (BERT + BiLSTM)](./NER)
* **Domain:** NLP / Information Extraction
* **Core Tech:** BERT Contextual Embeddings, Bi-Directional LSTM, Gradio.
* **Highlight:** Achieves a **95% F1-score** for Person (Name) detection. Features a web UI for real-time entity highlighting.

### 2. [Cyberbullying Detection System](./Cyberbullying-Detection)
* **Domain:** NLP / Sentiment Analysis
* **Core Tech:** SVM, CNN, LSTM, TF-IDF Vectorization.
* **Highlight:** A multi-class classifier trained on 47k+ tweets to identify specific types of online toxicity (Age, Ethnicity, Religion, etc.).

### 3. [Computer Vision Suite](./Image_Classification_Suite)
A collection of safety and sustainability focused vision models:
* **[Helmet Detection](./Image_Classification_Suite/HelmetDetectionYOLOv8):** Real-time Object Detection using **YOLOv8** (mAP@50: 0.92).
* **[Face Mask Detection](./Image_Classification_Suite/Face_Mask_Image_Classifcation):** Binary CNN classifier for public safety monitoring.
* **[Garbage Classification](./Image_Classification_Suite/Garbage_Classification):** Multi-class CNN for automated waste sorting.
